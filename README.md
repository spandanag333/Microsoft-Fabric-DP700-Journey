# Microsoft-Fabric-DP700-Journey

## About
This repository documents my hands-on learning journey while preparing for the Microsoft Certified: Fabric Data Engineer Associate (DP-700) certification.

The goal of this repository is to understand Microsoft Fabric from both a technical and business perspective by completing labs, documenting learnings, and building practical data engineering solutions.

## Technologies
- Microsoft Fabric
- OneLake
- Lakehouse
- Dataflow Gen2
- Data Pipelines
- Data Warehouse
- Spark
- SQL
- Power BI

## Learning Goals
- Understand Fabric Architecture
- Build end-to-end Data Engineering solutions
- Prepare for DP-700 Certification
- Improve Azure Data Engineering skills
